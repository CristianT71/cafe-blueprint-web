# Cafe Blueprint

Proyecto de desarrollo web enfocado en la implementación de interfaces responsivas utilizando Bootstrap.

---

## Objetivo

Aplicar conceptos de maquetación, sistema de grid y componentes de Bootstrap en el desarrollo de una interfaz para un establecimiento comercial.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- Bootstrap 4.1
